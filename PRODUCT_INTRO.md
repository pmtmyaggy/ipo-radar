# IPO-Radar 产品介绍

> **美股 IPO 投资决策情报系统** —— 从招股到上市，全程追踪，智能评分

---

## 一、产品定位

IPO-Radar 是一款面向美股投资者的 IPO 信息聚合与决策辅助工具。它自动追踪即将上市的新股，整合多维度数据（招股书、市场情绪、基本面、机构持仓），输出可量化的投资决策评分，帮助投资者在 IPO 窗口期做出更明智的选择。

**一句话价值**: *不再错过优质 IPO，不再踩雷劣质新股*

---

## 二、核心功能模块

### 📊 1. 一站式信息看板

| 功能 | 说明 | 用户价值 |
|------|------|----------|
| **IPO 日历** | 自动抓取未来 30 天即将上市的新股清单 | 提前锁定关注标的 |
| **S-1 招股书** | SEC EDGAR 官方文件，关键信息自动提取 | 省去翻阅冗长文档的时间 |
| **基本面分析** | 营收、利润、增长率、行业对比 | 3 分钟了解公司质地 |
| **价格追踪** | 上市首日及后续价格走势 | 观察市场真实反应 |

**使用场景**: 每周一打开 Dashboard，快速浏览本周有哪些 IPO，筛选出值得深入研究的目标。

---

### 🎯 2. 智能评分系统

综合 5 大维度，输出 0-100 分的投资决策评分：

```
评分维度及权重:
├── 基本面质量 (30%)      → 财务健康度、盈利能力
├── 市场热度   (25%)      → 认购倍数、社交媒体讨论量
├── 行业前景   (20%)      → 行业增速、竞争格局
├── 估值水平   (15%)      → 定价区间合理性
└── 机构背书   (10%)      → 承销商等级、基石投资者

输出结果:
• 80-100分: 强烈推荐
• 60-79分:  值得关注
• 40-59分:  中性观望
• 0-39分:   建议回避
```

**使用场景**: 面对多只同日上市的 IPO，先看评分排序，优先研究高分标的。

---

### 📡 3. 实时数据监控

**数据源覆盖**:
- 📈 **Yahoo Finance** - 实时股价、交易量、基本面数据
- 📄 **SEC EDGAR** - S-1 招股书、13F 机构持仓
- 📅 **Benzinga** - IPO 日历、定价信息

**更新频率**:
- 股价数据: 实时 (15分钟延迟)
- 招股书: 每小时扫描
- IPO日历: 每日更新
- 机构持仓: 季度更新

---

### 🏦 4. 机构持仓追踪 (13F)

追踪季度机构持仓变化：

| 指标 | 说明 |
|------|------|
| 新进机构 | 首次建仓的机构数量 |
| 退出机构 | 清仓离场的机构数量 |
| 持仓变化 | 总持仓量环比变化 |
| 前十大股东 | 持仓最多的机构名单 |

**使用场景**: 上市后 45 天（首个 13F 披露节点），查看是否有知名机构建仓，验证市场认可度。

---

### 🔔 5. 智能通知系统

**飞书/钉钉集成** - 关键信息自动推送：

```
通知场景:
├── 新股申购提醒    → "明日上市: ARM (ARM), 定价区间 $47-$51"
├── 定价确定通知    → "理想汽车定价 $11.5/股，高于上限 15%"
├── 首日表现报告    → "开盘涨 35%，成交量 5000 万"
├── 机构持仓更新    → "高瓴资本新进 500 万股，黑石减持 30%"
└── 系统异常告警    → "数据抓取异常，请检查网络"
```

**使用场景**: 绑定飞书群，重要 IPO 信息自动推送到手机，不再错过交易机会。

---

### 📈 6. 上市后追踪

IPO 只是开始，持续追踪才是关键：

- **锁定期监控** - 内部人股票解禁时间提醒
- **财报日历** - 上市后首份季报/年报披露时间
- **技术分析** - 价格突破、形态识别
- **情绪追踪** - 社交媒体情绪变化

**使用场景**: 持有 IPO 股票期间，持续接收关键事件提醒，辅助持仓决策。

## 三、典型用户场景

### 场景 A: 短线交易者

**需求**: 寻找首日有爆发力的新股

**IPO-Radar 方案**:
1. 筛选评分 70+ 的热门 IPO
2. 查看市场情绪（认购倍数、社交媒体热度）
3. 设定定价提醒，准备首日开盘交易
4. 接收首日表现报告，决定持仓过夜或离场

---

### 场景 B: 价值投资者

**需求**: 寻找长期持有的优质标的

**IPO-Radar 方案**:
1. 筛选基本面评分 80+ 的成长股
2. 研读 S-1 招股书关键章节
3. 上市后观察 3-6 个月价格走势
4. 查看机构持仓变化，跟随聪明钱建仓

---

### 场景 C: 机构投资者

**需求**: 批量跟踪 IPO Pipeline，生成研究报告

**IPO-Radar 方案**:
1. 导出未来 90 天 IPO 清单
2. 批量获取招股书 PDF 和关键数据
3. 对比同行业已上市公司的估值倍数
4. 生成标准化评分报告供投资决策

## 五、快速开始

### 5 分钟启动

```bash
# 1. 克隆项目
git clone https://github.com/pmtmyaggy/ipo-radar.git
cd ipo-radar

# 2. 配置环境变量（仅需一项）
cp .env.example .env
# 编辑 .env: 填写 EDGAR_IDENTITY="你的名字 你的邮箱@example.com"

# 3. 启动服务
make dashboard

# 4. 浏览器访问
open http://localhost:8501
```

## 六、技术亮点 (面向技术用户)

- **数据源可靠**: SEC EDGAR 官方 API、Yahoo Finance 实时数据
- **本地部署**: 数据存储在本地 SQLite，保护隐私
- **容器化**: Docker 一键部署，跨平台支持
- **可扩展**: 模块化架构，易于添加新数据源
- **开源**: 代码透明，可自定义评分算法
